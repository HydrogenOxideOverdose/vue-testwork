## Тестовое задание

Vue.js приложение для управления товарами с полным CRUD функционалом, валидацией и сохранением в LocalStorage.

### Таблица товаров

```
  Колонки: ID, Название, Категория, Цена, Количество, Статус, Действия
  Управление: Удаление, перемещение вверх/вниз
  Статус: "В наличии" (количество > 0) / "Нет в наличии" (количество = 0)
```

### Форма добавления

```
  Поля: Название, Категория, Цена, Количество
  Валидация:
  Название: обязательно, минимум 3 символа
  Цена: обязательно, число > 0
  Количество: обязательно, целое число >= 0
  Категория: обязательно
  Ошибки: Показываются при blur или submit
```

### Сохранение данных

   LocalStorage: Автосохранение и восстановление данных

   Кнопка "Сохранить": Активна только при изменениях

   Состояние: Сохраняется порядок товаров

### Mock API

   Задержка: 500-1500ms для имитации реального API

   Уведомления: Успех/ошибка с автоскрытием через 3 секунды

### Технологии

   Vue 2.6 + TypeScript

   Vue Class Component + Decorators


### Установка и запуск


```
# Клонировать репозиторий
git clone https://github.com/HydrogenOxideOverdose/vue-testwork
cd vue-testwork

# Установить зависимости
npm install

# Запуск development сервера
npm run serve

# Сборка для production
npm run build
```

Приложение будет доступно по адресу localhost с порта указаного в консоли

### Основные возможности
CRUD операции

   Create: Добавление новых товаров через форму

   Read: Отображение списка товаров в таблице

   Update: Перемещение товаров (изменение порядка)

   Delete: Удаление товаров с подтверждением

### Валидация форм

   Реалтайм валидация при вводе

   Показ ошибок при потере фокуса

   Блокировка отправки при ошибках

### Состояние интерфейса

   Loading состояния при API запросах

   Визуальная индикация загрузки

   Блокировка взаимодействия во время операций

### Уведомления

   Успешные операции (зеленые)

   Ошибки (красные)
